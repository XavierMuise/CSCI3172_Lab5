## Lab Questions

(a) a const variable is only accessible inside the block where it is declared, so for example if declared inside of an if block it would only be accesiible in that if block, not outside 
(b) let and const were added to javascript in 2015 with the addition of ECMAScript 2015 (ES6)
(c) It depends. let is accessible inside the block it was declared in. var is accessible in the function it was declared in. Since var has a wider scope, if youre reusing variable names it could use the wrong variable and cause bugs.
    Also, variables assigned with let cant be reassigned., so if you reassign your original var one there would be issues.
